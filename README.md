# Cloud-Based Secure Web Application

## Project Overview

This project is a Flask-based secure web application developed and deployed on AWS EC2. The application provides user registration, login authentication, role-based access control, file upload functionality, and activity logging.

## Features

- User Registration
- User Login Authentication
- Password Hashing
- Role-Based Access Control
- User Dashboard
- Admin Dashboard
- Activity Logging
- File Upload System
- MySQL Database Integration

## Technologies Used

- Python
- Flask
- MySQL
- Gunicorn
- Nginx
- AWS EC2
- Git & GitHub

## Installation

1. Clone the repository

git clone https://github.com/yourusername/cloud-secure-web-application.git

2. Install dependencies

pip install -r requirements.txt

3. Configure MySQL database

4. Run the application

python3 app.py

## AWS Deployment

The application is deployed on AWS EC2 using Gunicorn and Nginx as a reverse proxy server.
